Windwos or MacOS

## Setting before use

```
nano client_gui.py

ip_server = ''
```
In '' insert the ip address of the server.

__You must use the keys (.key, .crt) that are used for server.__

## Start
```
python3 client_gui.py
```

### Windows

Build py to exe

```
auto-py-to-exe
```
